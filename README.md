# QuickTodo - 极简桌面待办助手 📝

> 一款专注于桌面体验的轻量级待办事项应用，让任务管理变得简单高效

## 🎯 解决的痛点

针对**忙碌的职场人士和学生群体**，解决以下核心问题：
- ❌ 打开APP才能管理任务，操作繁琐
- ❌ 紧急任务容易被忽略
- ❌ 界面复杂，干扰专注

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 📱 **桌面小组件** | 无需打开APP，直接在桌面查看、添加、完成任务 |
| 🔴 **紧急标记** | 重要任务标红提醒，一目了然 |
| 🔄 **实时同步** | 桌面与APP数据双向同步，确保信息一致 |

## 🛠️ 技术栈

- **语言**：Kotlin
- **框架**：Android Jetpack (ViewModel, Room, Coroutines)
- **UI**：Material Design 3 + Jetpack Compose
- **架构**：MVVM + Repository 模式

## 📱 使用说明

### 添加任务
1. 点击桌面小组件右上角的 **+** 按钮
2. 输入任务内容
3. 勾选「紧急」标记重要任务（可选）
4. 点击「保存」完成添加

### 完成任务
- **桌面**：直接点击任务项即可切换完成状态
- **APP**：点击任务左侧圆形复选框

### 切换语言
1. 打开 APP
2. 点击右上角 ⚙️ 设置图标
3. 选择中文 / English

## 📷 界面展示

- **APP主界面**：简约ins风设计，支持任务的添加、完成、删除操作
- **桌面小组件**：白色圆角卡片样式，显示待办数量，紧急任务红色「!」标记
